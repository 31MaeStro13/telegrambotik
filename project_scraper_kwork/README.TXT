handlers/user.py
| Команда / Хендлер | Что делает |
|-------------------|------------|
| `/start` | Приветствие, показывает список команд |
| `/help` | Показывает справку по боту |
| `/favorites` | Показывает список избранных проектов пользователя |
| `/hidden` | Показывает список скрытых проектов |
| `/data` | Показывает последние 10 проектов (исключая скрытые) |
| `/stats` | Показывает статистику по базе проектов |
| `/profile` | Если профиль есть — показывает его с кнопкой "Редактировать", если нет — предлагает создать |
| `/edit_profile` | Запускает FSM для создания/редактирования профиля (первый вопрос) |
| `/cancel` (вне FSM) | Сообщает, что нечего отменять |
| `/cancel` (внутри FSM) | Прерывает заполнение профиля и очищает состояние |


handlers/profile.py

| Хендлер / Состояние | Что делает |
|---------------------|------------|
| `process_stack()` | Принимает текст (стек), проверяет длину, сохраняет, переводит в `experience` |
| `process_experience()` | Обрабатывает выбор опыта из кнопок, сохраняет, удаляет сообщение с кнопками, переводит в `goals` |
| `process_goals()` | Принимает текст (цели), проверяет, сохраняет, показывает подтверждение с кнопками, переводит в `confirm` |
| `process_save()` | Сохраняет профиль в БД, удаляет сообщение с подтверждением, отвечает результатом, очищает состояние |
| `process_restart()` | Удаляет сообщение и начинает заполнение заново (возврат в `stack`) |
| `warning_not_experience()` | Если в состоянии `experience` прислали текст вместо кнопок — предупреждение |
| `warning_not_confirm()` | Если в состоянии `confirm` прислали текст вместо кнопок — предупреждение |

 📌 Состояния FSM

| Состояние | Что хранит / ожидает |
|-----------|----------------------|
| `stack` | Текст — какой стек технологий |
| `experience` | Выбор из кнопок (0-1, 1-3, 3+) |
| `goals` | Текст — какие проекты ищет |
| `confirm` | Подтверждение данных (кнопки "сохранить" / "заново") |
